
# Minimalist Feedback API

A lightweight API for collecting and managing user feedback.

## Features

- Simple REST endpoints
- Minimal dependencies
- Easy to deploy

## Installation

```bash
git clone https://github.com/yourusername/minimalist-feedback-api.git
cd minimalist-feedback-api
npm install
```

## Usage

```bash
npm start
```

The API will be available at `http://localhost:3000`

## API Endpoints

- `POST /feedback` - Submit feedback
- `GET /feedback` - Retrieve feedback
